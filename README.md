## Получение строки для подключения к базе данных

Перейдите в "Databases"

![image](https://user-images.githubusercontent.com/66993983/151689657-41679de7-9e02-46fa-a9ab-280c0465ef94.png)

Нажмите "Connect"

![image](https://user-images.githubusercontent.com/66993983/151689668-56cd96f8-4d2b-4098-830c-e68a57ec9268.png)

Выберите "Connect your application"

![image](https://user-images.githubusercontent.com/66993983/151689680-eeb1e811-daf4-4356-bfb8-f078e86af2bd.png)

В пункте "Driver" выберите "Python", а в "Version" выберите "3.6 or later"

![image](https://user-images.githubusercontent.com/66993983/151689723-aae22001-25e8-4b54-8ed5-7d7c0b73def7.png)

Скопируйте строку для подключения, для этого нажмите на значок копирования

![image](https://user-images.githubusercontent.com/66993983/151689751-9fcaa5ec-7176-4b8d-8e5d-b205591d70da.png)

В этой строке есть имя и пароль от пользователя базы данных (Логин:Пароль), но пароль не вписан.

В этой строке, нужно заменить "<password>", на тот пароль, который вы делали для этого пользователя базы данных

Если имя пользователя базы данных не совпадает, то откройте блокнот и замените неправильное имя, на правильное.

![image](https://user-images.githubusercontent.com/66993983/151689840-c455ef71-c809-4c62-a34a-a0f62d4bd399.png) 
 
Отлично! Теперь у вас есть готовая строка для подключения к базе данных.
 
Как добавить базу данных в приложение, вы можете посмотреть [здесь](#добавление-базы-данных)

### Добавление экспоната

Чтобы добавить экспонат, нужно нажать кнопку "Добавить экспонат" на главном окне.

![image](https://user-images.githubusercontent.com/66993983/151671410-d42b5b14-674a-44d1-a318-5a1056c205ce.png)

В открывшемся окне выбираем:
* фонд музея
* подраздел фонда музея
* Тип фонда (Основной или научно-вспомогательный)
* Название (Обязательно должно быть уникальным, то есть не должн быть 2 экспоната с одинаковыми названиями)
* Номер учётной записи (Необязательно)

Дальше нажимаем кнопку "Далее"

![image](https://user-images.githubusercontent.com/66993983/151671605-e9e883fc-6620-481c-a005-719de4159d0c.png)

В открывшемся окне вводим:
* ФИО дарителя (Необязательно)
* Место хранения (Необязательно)
* Описание (Необязательно)

Нажимаем кнопку "Добавить"

![image](https://user-images.githubusercontent.com/66993983/151671679-1df14766-ab9f-4cc0-9006-83b2c6c11658.png)

## Создание геометрии

В папке  приложения есть папка “Resourses” с геометриями - “Geometry” и темами - “styles” соответственно. В этих папках находятся геометрии и стили которые отображаются в настройках для их применения. Их можно добавлять или убирать.

![image](https://user-images.githubusercontent.com/66993983/151687921-16bc680d-2588-4a5c-806d-de35c30001aa.png)
![image](https://user-images.githubusercontent.com/66993983/151687942-f7712d8e-ae7e-4270-b0d3-e2cf1c657f13.png)

Темы для приложения хранятся в виде css файла. Их можно редактировать и т.д. А вот с геометриями уже по-другому.

Сам файл геометрии хранится в формате ini на языке Lua. В файле находятся переменные с названием каждого объекта в приложении и их размерами в высоту и длину, местоположением, а также размером шрифта.

И так, как же сделать геометрию? Просто перейдите в папку "Resourses" --> "Доп прога".

+ В папке «OriginUI» находятся стандартные геометрии в формате ui.
+ С папкой «Workspace» программа будет взаимодействовать.
+ А в папке «Qt Designer» находится программа для редактирования ui файлов. 

![image](https://user-images.githubusercontent.com/66993983/151688107-da89a227-e535-4228-be2b-4e56d90c9602.png)

После этого откроется окно программы «Qt Designer» в которой нужно нажать «Open…», выбрать все файлы (или только некоторые) из папки «Workspace» и нажать «Открыть»

Чтобы начать «творить», нужно скопировать файлы из папки «OriginUI» в папку «Workspace», затем перейти в папку «Qt Designer» и открыть файл «designer».

После этого откроется окно программы «Qt Designer» в которой нужно нажать «Open…», выбрать все файлы (или только некоторые) из папки «Workspace» и нажать «Открыть» 

![image](https://user-images.githubusercontent.com/66993983/151688143-337d9aa8-3098-4c8c-8033-3e976d51b914.png)

![image](https://user-images.githubusercontent.com/66993983/151688169-082aad71-0c8a-4120-8531-6a3cfdd156a8.png)

Также можно открыть файлы ui нажав кнопку с папкой в левой верхней части программы

![image](https://user-images.githubusercontent.com/66993983/151688193-2d266c88-f33b-46cc-a54a-7b422e9512fd.png)

Теперь можно редактировать выбранные файлы.
Чтобы увеличить размер окна, нужно навести курсор на боковую часть редактируемого окна, зажать левую кнопку мыши и двигать курсор

![image](https://user-images.githubusercontent.com/66993983/151688404-5e2c0fff-744e-4c46-960b-a24538848293.png)

Чтобы изменять местоположение объекта, то нужно навести курсор мыши на этот объект, зажать левую кнопку мыши и передвинуть этот объект

![image](https://user-images.githubusercontent.com/66993983/151688419-4eb78843-b341-4167-b3fd-81b95e0071b4.png)

Чтобы изменить размер объекта, то нужно нажать на него правой кнопкой мыши, после чего появится выделение на этом объекте

![image](https://user-images.githubusercontent.com/66993983/151688430-12462ca1-ee2d-4bdb-88cd-76f7ca7d8049.png)

На выделении будут квадратики, которые нужны для изменения размера объекта. Нужно всего лишь зажать квадратик левой кнопкой мыши и начать двигать мышкой

![image](https://user-images.githubusercontent.com/66993983/151688439-806983c9-b99f-4d74-9f14-e7792ac070b6.png)

Чтобы изменить размер шрифта, то нужно нажать на нужный вам объект левой кнопкой мыши, после чего справа в колонке с характеристиками найти «font».

![image](https://user-images.githubusercontent.com/66993983/151688491-81528a2b-aa0b-4a58-ab2a-9f2f593dc1a8.png)

 Этот пункт является списком, который можно открыть с помощью стрелочки слева
 
 ![image](https://user-images.githubusercontent.com/66993983/151688499-59b9737d-41b2-48f8-9b56-ab8f3b0fc6a3.png)

Далее в появившимся пункте «Point Size» справа можно менять размер шрифта у данного объекта, но помните, что у всех кнопок должен быть одинаковый размер шрифта, который вы укажете позже

![image](https://user-images.githubusercontent.com/66993983/151688521-47c1a9e2-aa92-49f4-8ffb-94e09ac33a27.png)

Чтобы сохранить изменения нужно нажать на окно, которое хотим сохранить, а после этого нажать на кнопку сохранения в левом верхнем углу

![image](https://user-images.githubusercontent.com/66993983/151688538-144a71ed-b7ac-4f33-b369-7af0f4710fc0.png)

После того, как вы закончили редактировать, вы должны запустить файл «start», который находится в папке «Доп прога», ввести название для геометрии, нажать "Enter", ввести размер шрифта для всех кнопок и нажать "Enter"

![image](https://user-images.githubusercontent.com/66993983/151688580-e2f3ee05-0e19-4572-8209-0c44472eb5c5.png)

![image](https://user-images.githubusercontent.com/66993983/151688646-b8aec9c3-7a1f-49a9-b072-8e071bd3c95e.png)

После этого в папке с программой появится файл с введенным ранее названием

![image](https://user-images.githubusercontent.com/66993983/151688663-21f4ced5-249d-4712-9c24-bd99bccc1b77.png)

Дальше просто перемещаем этот файл в папку с геометриями

![image](https://user-images.githubusercontent.com/66993983/151688701-911284fd-90da-49aa-aaf1-f5c42ef944ae.png)

Готово, вы создали геометрию для приложения. Теперь вы можете выбрать её в настройках приложения
